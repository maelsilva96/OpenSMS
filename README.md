# OpenSMS

Aplicativo de SMS/MMS gratuito para Android

**Introdução**
-------------

O OpenSMS é um aplicativo de Android projetado para permitir a leitura e envio de SMS e MMS de forma gratuita. O objetivo do projeto é fornecer uma solução simples e fácil de 
usar para os usuários que desejam enviar mensagens de texto ou multimídia.

**Funcionalidades**
-----------------

* Envio de SMS/MMS de forma gratuita
* Leitura de SMS/MMS
* Suporte a diferentes tipos de dispositivos Android

**Requisitos Técnicos**
-----------------------

* API 24+ ou superior
* Linguagem Kotlin
* IDE Android Studio com plugin de Kotlin instalado

**Como usar o OpenSMS**
-------------------------

1. Baixe o aplicativo do Google Play Store.
2. Instale e abra o aplicativo.
3. Configure as suas preferências de envio de SMS/MMS.
4. Comience a enviar e receber mensagens.
